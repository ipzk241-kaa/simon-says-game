This document describes how the Simon Says web application collects, uses, and stores user data, as well as user rights under the **GDPR**.

---

## 1. General

1. This application is an educational project and is not intended for commercial use.
2. All user data is stored **locally** in the browser via **LocalStorage**. **Cookies are not used** to store games or personal data.
3. Using the application means agreeing to this privacy policy.

---

## 2. Data Collection and Use

- **LocalStorage:** only stores game progress and user settings (e.g. difficulty level, last score).
- **CookieConsent:** is only used to display the "Cookies" banner as required. The cookie stores information about whether the user clicked "Accept" or "Reject" and is not used for analytics.

**We do not collect:**
- IP addresses
- Personal data (name, email, phone)
- Device data or behavior outside the game

---

## 3. User rights (GDPR)

1. You have the right to **access** your data in LocalStorage.
2. You have the right to **delete** all game data from your browser.
3. You can **opt out** of the CookieConsent banner by clicking "Reject".

> The project does not have server-based data storage, so all data remains only in your browser.

---

## 4. Use and limitations

1. The application is intended for **educational purposes** and **personal use**.
2. It is prohibited to use the application for commercial purposes without the author's permission.
3. The author is not responsible for any data loss or other consequences of using the application.
