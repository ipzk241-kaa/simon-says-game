useGameLogic(colors, speed) → {Object}

Custom hook `useGameLogic` implements the "Simon Says" game logic. Responsibilities: - Storing the sequence of colors - Handling rounds and levels - Validating player clicks - Animating color highlights
Parameters:
Name Type Description
colors Array.<string> Array of available colors for the game.
speed number Animation speed for color highlights in milliseconds.
Properties:
Name Type Description
level number Current game level.
activeColor string | null Currently highlighted color.
isGameOver boolean Whether the game has ended.
startGame function Function to start the game.
handlePlayerClick function Handler function for player clicks. Receives a color string.

useGameSettings() → {Object}

Custom hook `useGameSettings` retrieves the current game settings from the Redux store. Responsibilities: - Getting the current difficulty level - Retrieving game parameters for the selected difficulty - Selecting the set of colors for the game based on `colorsCount`
Properties:
Name Type Description
settings Object Game settings for the current difficulty level.
colors Array.<string> Array of colors for the game according to the settings.

useSimonGame() → {Object}

Custom hook `useSimonGame` combines game settings and game logic for the "Simon Says" game. Uses: - `useGameSettings` to get current game settings (difficulty, speed, colorsCount) - `useGameLogic` to manage game state (sequence, levels, player clicks)
Properties:
Name Type Description
level number Current game level.

startGame function Function to start the game.

colors Array.<string> Array of colors for the game.
