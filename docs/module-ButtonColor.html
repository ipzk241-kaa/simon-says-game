Module: ButtonColor

`ButtonColor` is a component that renders a colored button on the game board. Responsibilities: - Rendering a button with a specific color - Highlighting the button when it is active - Triggering a callback when the button is clicked
Parameters:
Name Type Description
props.color string The button color (must match a CSS class name).
props.isActive boolean Indicates whether the button is currently highlighted.
props.onClick function Function called when the button is clicked.
